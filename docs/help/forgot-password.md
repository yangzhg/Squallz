# 密码忘了怎么办

## 先说结论

Squallz 没有后门，也不会保存可用于破解压缩包的秘密信息。忘记加密压缩包密码后，
通常不能恢复内容。

## 可以检查的地方

- 如果你曾在 Squallz 的密码弹窗里勾选“记住这个压缩包的密码”，可以重新打开该压缩包。
  Squallz 会尝试从系统密码库读取：
  - macOS：Keychain
  - Windows：Credential Manager
  - Linux：Secret Service
- 如果只是本次会话输入过密码，关闭应用后会话密码不会继续保存。
- 如果是别人发来的压缩包，只能向创建者索要正确密码。

## 不会发生的事

- Squallz 不会把明文密码写入 `settings.json`、localStorage、日志或任务历史。
- Squallz 不提供暴力破解功能。
- PAR2 恢复数据和 `.sqz` 内嵌恢复记录只能修复损坏或丢失的数据块，不能绕过加密密码。

## 建议

- 给长期归档使用密码管理器保存密码。
- 重要资料优先使用 `.sqz` 或外置 PAR2 增加损坏恢复能力，但仍要单独保存密码。
